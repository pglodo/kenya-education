Student Enrollment of Kenya

Taking a look at the difference in enrollment between male and female students, from grades 1 through 8 in the year 2014.

About

This web map was an excercise of map and User Interface design, by using our JavaScript and jQuery skills to dynamically display data based on user input. Additionally, dynamic 'info' boxes were created to display the quantified data and give the user a very smooth and satisfying experience.


Data acquired through Esri ArcGIS Online (link in footer).
